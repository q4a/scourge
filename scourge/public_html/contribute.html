Tech specs
S.C.O.U.R.G.E. uses SDL, OpenGL and GLUT. (It only uses GLUT to print text... hopefully this dependency will go away soon.) The code is written in C++.
I have compiled and ran it on Windows, Linux (Mandrake 9.2) and the Mac OS X.
