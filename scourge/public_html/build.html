Mac OS X
1. Get FINK and install the SDL libraries.
2. make -f Makefile.OSX
You should see some warnings, but it should compile. I also compiled with XCode, but I prefer not to use ide-s.

Mac OS X Notes:
-After compiling on the command line, I could only launch the app from the command line. (Not the Finder.) Note: this is fixed in CVS
-SDL or OpenGL seems to reverse the mouse pointer's Y coord. when running in windowed mode. I added code to work around this but I suspect this code will have to be removed in the future.

Linux
1. Make sure you have SDL, OpenGL and GLUT libraries installed.
2. make
I run Mandrake linux 9.2, so older versions of gcc may give you problems.

Linux notes:
-To check that you're running in hardware accelerated mode, type "glxinfo". Look for a line like "Direct rendering". If the value is "yes", you're ok. -Different drivers have spotty support for different video modes. Currently S.C.O.U.R.G.E. runs in 800x600 32bpp mode. You may want to change the code if your card doesn't support this mode.

Windows
1. Install MinGW and MSYS (In that order!)
2. Install GLUT for MinGW
3. Install SDL for windows
4. In MSYS, type: "make -f Makefile.win32"

Windows notes:
In general there should be no problems. You may encounter some opengl incompatibilities, but these will be fixed in future builds.
